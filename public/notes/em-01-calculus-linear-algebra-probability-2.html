Limits & Continuity

1.1   Limits — Definition & Evaluation Core

Limit — ε-δ Definition

lim_{x→a} f(x) = L means: for every ε > 0, there exists δ > 0 such that 0 < |x−a| < δ implies |f(x)−L| < ε. The function value at a doesn't matter — only the behaviour near a.

L'Hôpital's Rule

L'Hôpital's Rule

If lim f(x)/g(x) gives indeterminate form 0/0 or ∞/∞, and both f and g are differentiable near a:
lim_{x→a} f(x)/g(x) = lim_{x→a} f'(x)/g'(x)
Apply repeatedly if the result is still indeterminate. Only valid for 0/0 or ∞/∞ — not for 0·∞, ∞−∞, 1^∞, 0^0 directly (convert first).

Evaluate lim_{x→0} (sin x − x)/x³:

Form: 0/0. Apply L'Hôpital: → lim (cos x − 1)/(3x²) still 0/0 → lim (−sin x)/(6x) still 0/0 → lim (−cos x)/(6) = −1/6 Answer: −1/6 Alternative using Taylor: sin x = x − x³/6 + x⁵/120 − ... (sin x − x)/x³ = (−x³/6 + ...)/x³ = −1/6 + ... → −1/6 ✓

1.2   Continuity & Differentiability GATE Tested

Continuity — Three Conditions

f is continuous at x = a iff ALL THREE hold:
1. f(a) is defined.
2. lim_{x→a} f(x) exists (left limit = right limit).
3. lim_{x→a} f(x) = f(a).
A function continuous on a closed interval [a,b] satisfies the Extreme Value Theorem (attains max and min) and Intermediate Value Theorem.

Differentiability Implies Continuity

If f is differentiable at a, then f is continuous at a. The converse is false: |x| is continuous at 0 but not differentiable (corner). Continuity is necessary but not sufficient for differentiability.

f(x) = |x|. Differentiable at x=0?

Left derivative: lim_{h→0⁻} (|h| − 0)/h = lim h→0⁻ (−h)/h = −1 Right derivative: lim_{h→0⁺} (|h| − 0)/h = lim h→0⁺ h/h = 1 Left ≠ Right → NOT differentiable at 0. Continuous at 0 (limit = 0 = f(0)) but not differentiable. ✓

2.2   Mean Value Theorems GATE Tested

Rolle's Theorem

If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then there exists c ∈ (a,b) such that f'(c) = 0.
Geometric meaning: there is at least one horizontal tangent between two points at the same height.

Mean Value Theorem (Lagrange's MVT)

If f is continuous on [a,b] and differentiable on (a,b), then there exists c ∈ (a,b) such that:
f'(c) = [f(b) − f(a)] / (b − a)
Geometric meaning: the instantaneous rate of change at c equals the average rate of change over [a,b]. Rolle's theorem is the special case f(a)=f(b).

Cauchy's Mean Value Theorem

If f and g are continuous on [a,b] and differentiable on (a,b), and g'(x) ≠ 0 on (a,b):
[f(b)−f(a)] / [g(b)−g(a)] = f'(c)/g'(c) for some c ∈ (a,b).
Setting g(x)=x recovers Lagrange's MVT.

Chapter 04
Multivariable Calculus — Partial Derivatives & Optimization

4.1   Partial Derivatives & Gradient GATE Tested

Partial Derivative

∂f/∂x: differentiate f(x,y,...) treating all other variables as constants. ∂f/∂y: differentiate treating x (and others) as constants.
The gradient ∇f = (∂f/∂x, ∂f/∂y) points in the direction of steepest increase of f.

f(x,y) = x²y + sin(xy). Find ∂f/∂x and ∂f/∂y.

∂f/∂x = 2xy + y·cos(xy) (treat y as constant) ∂f/∂y = x² + x·cos(xy) (treat x as constant)
Chain Rule for Partial Derivatives

If z = f(x,y) and x = x(t), y = y(t):
dz/dt = (∂f/∂x)(dx/dt) + (∂f/∂y)(dy/dt)
If z = f(x,y), x = x(s,t), y = y(s,t):
∂z/∂s = (∂f/∂x)(∂x/∂s) + (∂f/∂y)(∂y/∂s)

Critical Points & Second-Order Test

Second Derivative Test for f(x,y)

At a critical point (∂f/∂x = 0, ∂f/∂y = 0), compute the Hessian determinant:
D = f_{xx}·f_{yy} − (f_{xy})²
If D > 0 and f_{xx} > 0: local minimum.
If D > 0 and f_{xx} < 0: local maximum.
If D < 0: saddle point.
If D = 0: inconclusive.

Maxima and Minima — Lagrange Multipliers

To optimize f(x,y) subject to constraint g(x,y) = c:
Solve: ∇f = λ∇g and g(x,y) = c simultaneously.
The Lagrange multiplier λ represents the rate at which the optimal value of f changes with c.

Chapter 06
Systems of Equations & Rank

6.1   Rank & Consistency of Linear Systems GATE Critical

Rank of a Matrix

The rank of matrix A is the number of linearly independent rows (= number of linearly independent columns). It equals the number of non-zero rows in the row-echelon form. rank(A) ≤ min(m, n) for an m×n matrix.

Consistency Conditions for Ax = b

Let A be m×n, [A|b] be the augmented matrix.
No solution (inconsistent): rank(A) ≠ rank([A|b]).
Unique solution: rank(A) = rank([A|b]) = n (full column rank).
Infinitely many solutions: rank(A) = rank([A|b]) < n.
Number of free variables = n − rank(A).

Classify solutions of the system:

A = [1 2 3; 2 4 6; 1 2 4], b = [6; 12; 7] Row reduce [A|b]: R2 ← R2 − 2R1: [1 2 3 | 6] [0 0 0 | 0] [1 2 4 | 7] R3 ← R3 − R1: [1 2 3 | 6] [0 0 0 | 0] [0 0 1 | 1] rank(A) = 2, rank([A|b]) = 2, n = 3. rank(A) = rank([A|b]) = 2 < 3 → infinitely many solutions. Free variables = 3 − 2 = 1 (x₂ is free). From R3: x₃ = 1. From R1: x₁ + 2x₂ + 3(1) = 6 → x₁ = 3 − 2x₂. Solution: x = [3, 0, 1] + t[−2, 1, 0] for any t ∈ ℝ.
Homogeneous System Ax = 0

Ax = 0 always has the trivial solution x = 0. It has a non-trivial solution iff rank(A) < n iff det(A) = 0 (for square A). The solution space is called the null space of A, with dimension = n − rank(A) (nullity).
Rank-Nullity Theorem: rank(A) + nullity(A) = n (number of columns).

Eigenvalues & Eigenvectors

7.1   Eigenvalues — Computation & Properties GATE Critical

Eigenvalue Equation

λ is an eigenvalue of A and x (≠0) is the corresponding eigenvector if: Ax = λx.
Equivalently: (A − λI)x = 0 has a non-trivial solution.
This happens iff det(A − λI) = 0 — the characteristic equation.

Find eigenvalues of A = [[3, 1], [1, 3]]:

det(A − λI) = det([[3−λ, 1], [1, 3−λ]]) = (3−λ)² − 1 = 9 − 6λ + λ² − 1 = λ² − 6λ + 8 = (λ−2)(λ−4) = 0 Eigenvalues: λ₁ = 2, λ₂ = 4 Check: trace(A) = 3+3 = 6 = λ₁+λ₂ = 2+4 = 6 ✓ det(A) = 9−1 = 8 = λ₁·λ₂ = 2·4 = 8 ✓ For λ=2: (A−2I)x = 0 → [[1,1],[1,1]]x = 0 → x₁+x₂=0 → x = t[1,−1] For λ=4: (A−4I)x = 0 → [[−1,1],[1,−1]]x = 0 → x = t[1,1]

Cayley-Hamilton Theorem

Every square matrix satisfies its own characteristic equation. If p(λ) = det(A−λI) = 0 is the characteristic polynomial, then p(A) = 0. Uses: compute Aᵏ for large k by reducing via the characteristic equation. Compute A⁻¹ without Gaussian elimination.

If char poly of A is λ² − 5λ + 6 = 0, find A⁻¹ using Cayley-Hamilton:

By Cayley-Hamilton: A² − 5A + 6I = 0 → A² = 5A − 6I → 6I = 5A − A² → I = (5A − A²)/6 → A⁻¹ = A⁻¹·I = A⁻¹·(5A−A²)/6 = (5I − A)/6

Chapter 08
Probability Theory — Axioms, Conditional & Bayes

8.1   Probability Axioms & Rules Core

Kolmogorov Axioms

For a sample space Ω and event A ⊆ Ω:
1. P(A) ≥ 0 for all events A.
2. P(Ω) = 1.
3. For mutually exclusive events A and B (A∩B=∅): P(A∪B) = P(A) + P(B).
These three axioms imply all probability rules.

RuleFormula
ComplementP(Ā) = 1 − P(A)
UnionP(A∪B) = P(A) + P(B) − P(A∩B)
Union (mutually exclusive)P(A∪B) = P(A) + P(B) when A∩B=∅
ConditionalP(A|B) = P(A∩B)/P(B), B≠∅
MultiplicationP(A∩B) = P(A|B)·P(B) = P(B|A)·P(A)
IndependenceA⊥B iff P(A∩B) = P(A)·P(B) iff P(A|B) = P(A)
Total ProbabilityP(A) = Σ P(A|Bᵢ)·P(Bᵢ) for partition {Bᵢ}
Bayes' Theorem

P(Bᵢ|A) = P(A|Bᵢ)·P(Bᵢ) / Σⱼ P(A|Bⱼ)·P(Bⱼ)
Updates prior probability P(Bᵢ) to posterior P(Bᵢ|A) after observing evidence A.
Terminology: P(Bᵢ) = prior, P(A|Bᵢ) = likelihood, P(Bᵢ|A) = posterior, Σ P(A|Bⱼ)P(Bⱼ) = evidence.

Classic Bayes: Disease testing. P(D)=0.01 (disease prevalence). Test: P(+|D)=0.99, P(+|Dᶜ)=0.05. Given positive test, what is P(D|+)?

P(+) = P(+|D)·P(D) + P(+|Dᶜ)·P(Dᶜ) = 0.99×0.01 + 0.05×0.99 = 0.0099 + 0.0495 = 0.0594 P(D|+) = P(+|D)·P(D) / P(+) = (0.99×0.01) / 0.0594 = 0.0099 / 0.0594 ≈ 0.167 (≈ 16.7%) Counterintuitive: despite 99% sensitive test, only 16.7% chance of disease given positive result — because the disease is rare (prior = 1%).

Chapter 09
Random Variables & Distributions

9.1   Random Variables — Expectation & Variance GATE Critical

Discrete Random Variable

X takes countable values. PMF (Probability Mass Function): P(X=x) for each x.
E[X] = Σ x·P(X=x) (expected value / mean)
E[g(X)] = Σ g(x)·P(X=x)
Var(X) = E[X²] − (E[X])²
SD(X) = √Var(X)

Continuous Random Variable

X takes values in an interval. PDF (Probability Density Function): f(x) with ∫_{−∞}^∞ f(x)dx = 1.
P(a ≤ X ≤ b) = ∫_a^b f(x)dx
E[X] = ∫_{−∞}^∞ x·f(x)dx
Var(X) = ∫_{−∞}^∞ x²·f(x)dx − (E[X])²

PropertyFormula
Linearity of expectationE[aX + b] = aE[X] + b
Expectation of sumE[X+Y] = E[X] + E[Y] (always, no independence needed)
Variance formulaVar(X) = E[X²] − (E[X])²
Variance of linear transformVar(aX + b) = a²Var(X)
Variance of sum (independent)Var(X+Y) = Var(X) + Var(Y) if X⊥Y
CovarianceCov(X,Y) = E[XY] − E[X]E[Y]
If independentE[XY] = E[X]·E[Y], Cov(X,Y) = 0

Poisson — Approximation to Binomial

When n is large and p is small with np = λ moderate, Binomial(n,p) ≈ Poisson(λ). Rule of thumb: n≥20 and p≤0.05. The Poisson PMF is simpler to compute than Binomial for large n. Both have mean λ = np, but Poisson has variance = mean = λ (while Binomial has variance = np(1−p) ≈ np = λ for small p).

Memoryless Property

The Geometric distribution (discrete) and the Exponential distribution (continuous) are the ONLY distributions with the memoryless property:
P(X > s+t | X > s) = P(X > t)
"Given that the event hasn't happened in the first s units, the probability of waiting t more units is the same as starting fresh." Used in: queuing theory, reliability analysis, network packet arrivals.

Central Limit Theorem (CLT)

If X₁, X₂, ..., Xₙ are i.i.d. random variables with mean μ and variance σ², then as n → ∞:
(X̄ − μ) / (σ/√n) → Normal(0, 1)
The sample mean X̄ = (X₁+...+Xₙ)/n is approximately Normal regardless of the distribution of individual Xᵢ. The approximation is good for n ≥ 30 in practice.

10.1   All the Traps in One Place

#TrapCorrect Understanding
1L'Hôpital applies to 0·∞L'Hôpital only applies to 0/0 or ∞/∞. For 0·∞, rewrite: f·g = f/(1/g) or g/(1/f) to get 0/0 or ∞/∞, then apply. Never apply L'Hôpital to 1^∞, 0^0, ∞−∞ directly.
2Continuity implies differentiabilityDifferentiability implies continuity (the stronger implies the weaker). Continuity does NOT imply differentiability. |x| is continuous everywhere but not differentiable at 0. Corners and cusps are continuous but not differentiable.
3det(cA) = c·det(A)det(cA) = cⁿ·det(A) for an n×n matrix (scalar multiplies EVERY row, each row multiplication multiplies det by c, so n rows give cⁿ). For c=2, n=3: det(2A) = 8·det(A), not 2·det(A).
4trace(AB) = trace(A)·trace(B)trace(AB) = trace(BA) — cyclic property. But trace(AB) ≠ trace(A)·trace(B) in general. Also: det(AB) = det(A)·det(B) — but the same does NOT hold for trace.
5A square system always has a unique solutionA square system Ax=b has: unique solution if det(A)≠0, no solution if det(A)=0 and rank([A|b])>rank(A), infinitely many if det(A)=0 and rank([A|b])=rank(A). Square doesn't guarantee unique.
6Sum of eigenvalues = determinantSum of eigenvalues = TRACE. Product of eigenvalues = DETERMINANT. These are the two fundamental eigenvalue identities. Mixing them up causes wrong answers in eigenvalue problems.
7Independent events means mutually exclusiveIndependent: P(A∩B) = P(A)P(B). Mutually exclusive: P(A∩B) = 0. If P(A)>0 and P(B)>0 and they are mutually exclusive, they CANNOT be independent (since P(A∩B)=0 ≠ P(A)P(B)>0). Independence and mutual exclusivity are nearly opposite concepts for non-trivial events.
8E[X²] = (E[X])²E[X²] = Var(X) + (E[X])². The equality E[X²] = (E[X])² holds only when Var(X)=0 (constant random variable). For any non-trivial X, E[X²] > (E[X])². Var(X) = E[X²]−(E[X])² ≥ 0 always.
9Poisson mean ≠ variancePoisson is the unique discrete distribution where mean = variance = λ. This is a characteristic property, not a coincidence. GATE tests: "For Poisson(λ), what is the variance?" → λ.
10Geometric distribution counts failuresTwo versions exist: X = number of trials until first success (support 1,2,..., mean=1/p), or X = number of failures before first success (support 0,1,..., mean=(1−p)/p). GATE specifies which — read carefully. Both have the memoryless property.
11∫_{−∞}^∞ x f(x)dx always convergesThe expectation E[X] = ∫x f(x)dx may not exist if the integral doesn't converge absolutely. Classic example: Cauchy distribution has no defined mean. GATE won't use such pathological cases but it's important conceptually.
12Rank-Nullity: rank + nullity = number of rowsRank + Nullity = number of COLUMNS, not rows. For m×n matrix: rank(A) + nullity(A) = n. Nullity = dimension of null space = number of free variables = n − rank.

10.3   GATE Question Patterns

  • Limit NAT: "Find lim_{x→0} (1−cos x)/x²." Use Taylor: (1−(1−x²/2+...))/x² = (x²/2−...)/x² = 1/2. 1-mark.
  • Determinant NAT: "If det(A) = 4 for a 3×3 matrix, find det(2A)." 2³×4 = 32. 1-mark trap.
  • Eigenvalue NAT: "Matrix has eigenvalues 2, 3, 5. Find det(A) and trace(A)." det=30, trace=10. 1-mark each.
  • Consistency MCQ: "For Ax=b with rank(A)=2, rank([A|b])=2, n=4. How many solutions?" Infinitely many (4−2=2 free variables). 1-mark.
  • Bayes NAT: Standard disease/test Bayes calculation. Apply formula P(D|+) = P(+|D)P(D)/P(+). 2-mark calculation.
  • Expectation NAT: "X is Binomial(10, 0.3). Find E[2X+1]." E[X]=10×0.3=3. E[2X+1]=2×3+1=7. 1-mark.
  • Variance NAT: "X is Poisson(4). Find Var(3X−2)." Var(X)=4. Var(3X−2)=9×4=36. 1-mark.
  • Integration NAT: "∫_0^1 x·eˣ dx." Integration by parts: [xeˣ]_0^1 − ∫_0^1 eˣdx = e − [eˣ]_0^1 = e−(e−1) = 1. 2-mark.
  • Conditional probability NAT: Two dice thrown. Given sum is odd, probability both are odd? Find P(both odd ∩ sum odd) / P(sum odd). Both odd → sum even. So P=0. "Can two odd numbers sum to odd?" 1-mark concept trap.
  • Taylor series application: "Expand f(x)=eˣsin x up to x³ terms." Multiply series: (1+x+x²/2+x³/6+...)(x−x³/6+...) = x+x²+x³/2−x³/6+... = x+x²+x³/3. 2-mark.

11.1   Measures of Central Tendency & Dispersion GATE Tested

Central Tendency — Three Measures

Mean (μ): Arithmetic average. μ = (Σxᵢ)/n. Sensitive to outliers — one extreme value pulls the mean significantly.
Median: Middle value when data is sorted. For n values: if n is odd, median = ((n+1)/2)th value. If n is even, median = average of (n/2)th and (n/2+1)th values. Robust to outliers.
Mode: Most frequently occurring value. A dataset may have no mode, one mode (unimodal), or multiple modes (bimodal/multimodal). Not affected by outliers.

Dispersion — Spread of Data

Range: Max − Min. Simple but very sensitive to outliers.
Variance (σ²): σ² = (Σ(xᵢ − μ)²)/n (population) or Σ(xᵢ − x̄)²/(n−1) (sample, Bessel's correction).
Standard Deviation (σ): √Variance. In the same units as data — directly interpretable.
IQR (Interquartile Range): Q3 − Q1 (75th percentile − 25th percentile). Most robust dispersion measure — unaffected by outliers.

Dataset: {2, 4, 4, 4, 5, 5, 7, 9}. Compute mean, median, mode, variance, SD:

n = 8 values Mean: (2+4+4+4+5+5+7+9)/8 = 40/8 = 5 Median: n=8 (even) → avg of 4th and 5th = (4+5)/2 = 4.5 Mode: 4 (appears 3 times, most frequent) Variance: Σ(xᵢ−5)²/8 (2−5)²=9, (4−5)²=1×3=3, (5−5)²=0×2=0, (7−5)²=4, (9−5)²=16 σ² = (9+3+0+4+16)/8 = 32/8 = 4 SD: σ = √4 = 2 Interpretation: data values typically differ from the mean by ±2 units.
Empirical Relationship (Unimodal Skewed Distribution)

For moderately skewed distributions: Mode ≈ 3×Median − 2×Mean. This relationship lets you find one central tendency measure from the other two. Also: if Mean > Median > Mode → positively skewed (right tail). If Mean < Median < Mode → negatively skewed (left tail). Symmetric → Mean = Median = Mode.

11.2   Correlation & Linear Regression GATE Tested

Pearson Correlation Coefficient (r)

Measures the strength and direction of the linear relationship between two variables X and Y.
r = Σ(xᵢ−x̄)(yᵢ−ȳ) / √[Σ(xᵢ−x̄)² · Σ(yᵢ−ȳ)²] = Cov(X,Y) / (σₓ·σᵧ)
Range: −1 ≤ r ≤ 1.
r = 1: perfect positive linear correlation. r = −1: perfect negative. r = 0: no linear correlation (but may have nonlinear relationship).

Simple Linear Regression — Least Squares Line

Fits the line ŷ = β₀ + β₁x that minimizes Σ(yᵢ − ŷᵢ)² (sum of squared residuals).
Slope: β₁ = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)² = Cov(X,Y)/Var(X) = r·(σᵧ/σₓ)
Intercept: β₀ = ȳ − β₁·x̄
The regression line always passes through (x̄, ȳ) — the mean point.

Data: (1,2), (2,4), (3,5), (4,4), (5,5). Fit regression line:

x̄ = 3, ȳ = 4 Σ(xᵢ−x̄)(yᵢ−ȳ) = (1−3)(2−4)+(2−3)(4−4)+(3−3)(5−4)+(4−3)(4−4)+(5−3)(5−4) = (−2)(−2)+(−1)(0)+(0)(1)+(1)(0)+(2)(1) = 4+0+0+0+2 = 6 Σ(xᵢ−x̄)² = 4+1+0+1+4 = 10 β₁ = 6/10 = 0.6 β₀ = 4 − 0.6×3 = 4 − 1.8 = 2.2 Regression line: ŷ = 2.2 + 0.6x Prediction at x=6: ŷ = 2.2+3.6 = 5.8
Correlation ≠ Causation

A high correlation (r near ±1) does not imply that X causes Y. Both may be caused by a third confounding variable. A classic example: ice cream sales and drowning deaths are positively correlated — both increase in summer (common cause: heat). GATE tests this conceptually: "What does r = 0.95 imply?" → Strong linear relationship, NOT causation.

11.3   Sampling Distributions & Hypothesis Testing Basics GATE Tested

Sampling Distribution of the Mean

If we take samples of size n from a population with mean μ and variance σ²:
Mean of sampling distribution: E[X̄] = μ
Variance of sampling distribution: Var(X̄) = σ²/n
Standard Error (SE) = σ/√n
By the Central Limit Theorem: X̄ ~ N(μ, σ²/n) for large n (≥ 30), regardless of population distribution.

Z-score Standardization

Z = (X − μ)/σ converts a value X to the number of standard deviations from the mean. The standard normal N(0,1) has mean 0, variance 1.
Key z-scores: z = 1.645 → 90% of data within ±1.645σ of mean (two-tailed). z = 1.96 → 95%. z = 2.576 → 99%.
For sample mean: Z = (X̄ − μ) / (σ/√n)

Hypothesis Testing Framework

H₀ (Null hypothesis): The default assumption (e.g., μ = μ₀, no difference).
H₁ (Alternative hypothesis): What we're trying to show (μ ≠ μ₀, or μ > μ₀, or μ < μ₀).
p-value: Probability of observing data this extreme if H₀ is true. If p-value < α (significance level, typically 0.05): reject H₀.
Type I error (α): Reject H₀ when it's true (false positive).
Type II error (β): Fail to reject H₀ when H₁ is true (false negative).
Power = 1 − β: probability of correctly rejecting a false H₀.
